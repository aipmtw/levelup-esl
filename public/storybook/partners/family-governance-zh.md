# 家庭治理 — Amy 是 Mark 的女兒

## 情況
三位主合作夥伴之一，**Amy**，是 Mark 的女兒。她的兒子（Mark 的外孫／孫子）是平台的第一位真實終端使用者。這是優勢 — 家人的信任無可匹敵、吃自家狗糧的回饋即時、情感產品故事真實。同時也產生了必須**明確處理而非默默迴避**的邊界。

這份文件寫出來，是為了讓 Audrey 或 Arita 如果有天讀到，可以看見這段關係、看見護欄、不會有默默偏袒讓她們感覺自己是二等。

## 原則

### 1. 家庭關係公開揭露，不藏
- Amy 在 `partners/amy-zh.md` 中明確被識別為「Mark 的女兒」。
- 其他主合作夥伴知道 Amy 是家人。
- 日後沒有驚訝。

### 2. 與其他主合作夥伴相同的平台條款
- Amy 以與 Audrey、Arita 相同的流程 onboarding。
- 同樣的 CMS、同樣的私人工作區、同樣的 PWA 工具、同樣的審閱關卡、同樣的 Luce credits。
- 沒有平台捷徑、沒有隱藏管理功能、沒有「老闆女兒」的特殊 code branch。
- **內部規則：** 如果未來有合作夥伴問「Amy 有沒有 X」，答案必須是「Amy 的待遇就跟你一樣」。

### 3. 計費是真實的，即使是家人
- Amy 與任何主合作夥伴一樣按計費方式收費 — 每次生成基礎設施費 + 分級，見 [`../pricing/partner-terms-zh.md`](../pricing/partner-terms-zh.md)。
- Mark 可以選擇**對她的帳號提供 credit**（家庭贈禮，不是價格折扣）。發票仍然顯示真實金額，這樣使用量資料與等級升級訊號才是真的。
- Mark **不可以**讓 Amy「帳外運作」 — 沒有不留軌跡的非正式優惠。那會在 Amy 大到要帶自己的子合作夥伴上自己的計費時造成混亂。
- **檢驗方式：** 如果 Audrey 看到 Amy 的帳號，她會覺得被公平對待嗎？保持答案是 yes。

### 4. Mark 不偷看 Amy 的私人工作區
- Mark 在平台上有 `global_admin` 覆寫權 — 技術上他可以看到每一位合作夥伴的私人工作區。
- **對 Amy 特別地**，Mark 承諾與對 Audrey 和 Arita 一樣的準則：只為合理的支援原因使用全域管理權，絕不隨意瀏覽。
- **明確禁止（即使技術上可行）：**
  - 未經 Amy 明確要求，瀏覽外孫的書。
  - 出於好奇閱讀 Amy 的客戶清單或定價備註。
  - 查看 Amy 的審閱備註「看看她怎麼想」。
- **明確允許（在 Amy 同意下）：**
  - Amy 拿一本特定的書給 Mark 看（「爸，第 3 頁是不是怪怪的？」）。
  - Amy 明確請 Mark 幫忙排除她工作區的問題。
  - Amy 聯絡不上、她的子合作夥伴需要協助的緊急支援。
- 每一次對 Amy 工作區的全域管理存取都會**被記錄**，Amy 可以在自己的工作區看到紀錄。

### 5. 外孫的內容是 Amy 的，不是 Mark 的
- 為 Mark 外孫生成的繪本是 Amy 的客戶資料，不是家族共有財產。
- Mark **不會**未經 Amy 明確同意就公開貼出、分享或用於行銷 — 即使得到同意，外孫的真實名字、臉孔、辨識特徵也應被遮蔽，除非 Amy 特別選擇開放。
- 與平台上任何其他家長的孩子一樣的規則。家人身分不能抹消隱私。

### 6. 產品決策的利益衝突
- Amy 的回饋很寶貴，但她的聲音不比 Audrey 或 Arita 大聲。
- 影響所有合作夥伴的產品決策走相同的流程，不論是誰提的。
- 若 Amy 要求一個只幫到她 0–3 分級、代價是 4–12 分級的功能，Mark 套用與任何合作夥伴一樣的「這是否服務整個平台」測試。
- **軟規則：** 當 Amy 倡議某件事時，Mark 有意識地問自己「如果 Audrey 要求同一件事，我會答應嗎？」。若答案是不會，那答案就是不會。

### 7. 商業邊界與家庭邊界分開
- 關於生意的分歧透過**商業管道**解決（合作夥伴協議、清楚的條款、書面決定），不走家庭動力。
- 若 Amy 想退出合作夥伴，她乾淨地退出 — Mark 不用情感勒索。平台存取權與客戶資料由她保留。
- 若 Mark 有一天想終止 Amy 的合作夥伴存取權（想不到什麼情境，但為完整起見），走與任何其他合作夥伴相同的流程。
- 家庭假日就是家庭假日。不要用星期天的午餐指示平台路線圖的改動。

### 8. 未來的家庭合作夥伴
- 如果 Mark 的太太、其他子女、兄弟姊妹等以後也想成為合作夥伴，他們走相同的流程。以上原則對所有人適用。
- 計畫中永遠不該有「創辦人家庭特殊等級」這種概念。

## 這對 Mark 換來什麼

- **與非家庭合作夥伴的乾淨關係。** Audrey 與 Arita 看計畫能看到 Amy 被公平對待。沒有耳語。
- **與女兒的乾淨關係。** 商業分歧不毒害家庭。家庭議題不扭曲生意。
- **真實而非戲劇化的狗糧回路。** Amy 給誠實的回饋，因為她信任邊界是清楚的、她沒有被看著。
- **一個 Mark 可以坦然說出的故事。**「我女兒是我第一位合作夥伴，我外孫是我第一位使用者，而我是這樣處理這段關係的」是強而有力的創辦人敘事。偷偷摸摸會削弱它。
