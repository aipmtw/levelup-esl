# M2 — 第二個里程碑

M1 驗證引擎與前兩位合作夥伴之後才進行。

## 交付項目

1. **EPUB3 匯出** — M1 之後的快速跟進。
   - Media overlay（SMIL）做文字—音訊同步朗讀。
   - 正統電子書閱讀器支援。

2. **自助計費**
   - 自動使用量計量。
   - 合作夥伴計費入口（查看使用量、付款、管理方案）。
   - 取代 M1 的手動月結。

3. **Flux + 每角色 LoRA**
   - 為需要招牌角色的合作夥伴提供嚴格主角一致性。
   - 從種子生成圖訓練 LoRA。
   - Pro 方案加購。

4. **Arita 成人課程系列**
   - 多堂課程結構。
   - 進度／分數追蹤。
   - **可下載課程包** — 成人學習者版的離線素材包。

5. **Premium 語音加購上線** — ElevenLabs v3 作為可由合作夥伴啟用的「Pro 朗讀」加購。

6. **合作夥伴頁面升級**
   - 客製子網域／白牌。
   - 多作者 CMS。
   - 更豐富的分析儀表板。

7. **多裝置 PWA 同步**
   - 有網路時同步閱讀進度。
   - 出版新版次時在背景刷新素材。
