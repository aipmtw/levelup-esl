# Questionnaire templates

Base questionnaires shipped by the platform. Partners fork and customize.

## Kid book — base questionnaire
1. Child's name.
2. Child's age (drives tier: 0–3 / 4–6 / 7–9 / 10–12).
3. Pick 1–3 interests (chips): dinosaurs, ice cream, spaceships, ninjas, kittens, LEGOs, unicorns, robots, cars, princesses, pirates, dragons…
4. Theme / goal (optional): bedtime, courage, sharing, counting, friendship…
5. Language mix: zh-TW (always on) + `[ ] en` + `[ ] ja`.
6. Art style (from partner's forked set).
7. Narration voice (from partner's forked set).

## English tutor — base intake (Arita)
1. Learner goal: TOEIC / business / professional domain / general.
2. If professional: which domain (IT+AI, finance, medical, legal, …).
3. Current level: A1 / A2 / B1 / B2 / C1.
4. Focus this session: reading / listening / speaking / grammar / vocab / role-play.
5. Native language (for bilingual explanations): zh-TW / ja-JP / other.

## Partner customization
- Add / remove fields.
- Rename labels per brand voice.
- Translate to any of the supported UI languages.
- Pre-fill from partner-specific defaults (e.g. Audrey defaults language mix to en+zh-TW).
