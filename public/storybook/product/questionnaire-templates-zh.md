# 問卷模板

平台出貨的基礎問卷。合作夥伴可分支客製。

## 兒童繪本 — 基礎問卷
1. 孩子名字。
2. 孩子年齡（決定分級：0–3 / 4–6 / 7–9 / 10–12）。
3. 選 1–3 個興趣（chips）：恐龍、冰淇淋、太空船、忍者、小貓、樂高、獨角獸、機器人、車子、公主、海盜、龍…
4. 主題／目標（選填）：睡前、勇氣、分享、數數、友誼…
5. 語言組合：繁中（恆常）+ `[ ] en` + `[ ] ja`。
6. 插畫風格（從合作夥伴分支的風格組）。
7. 朗讀聲音（從合作夥伴分支的聲音組）。

## Arita 家長諮詢問卷（老師通路專用）
Arita 在產生繪本前寄給學生家長填寫：
1. 學生目前英語程度（A1 / A2 / B1 / B2）。
2. 本書的學習目標（字彙主題、文法重點、閱讀理解、建立自信…）。
3. 想強調／避免的事項（飲食禁忌、宗教、家長偏好話題…）。
4. 希望包含的字彙清單（選填）。
5. 家長對難度／語氣的偏好。
6. Arita 可加上自己的教學筆記。

## 合作夥伴客製化
- 新增／移除欄位。
- 依品牌語氣重新命名標籤。
- 翻譯成任何支援的 UI 語言。
- 以夥伴特定預設值預填（例如 Audrey 預設語言為 en + zh-TW）。
