# 圖像生成

## M1 — gpt-image-1 + style-lock
- **模型：** gpt-image-1。
- **一致性：** 「style lock」prompt 範本 + **每本一張參考圖**（主角），在整本 10–20 頁中重複使用。
- **文字在圖外** — **不要**把頁面文字烤進圖裡。渲染器（HTML → PDF）在圖上或旁邊疊上文字。這讓多語版本變成微不足道（同一張圖、三個文字層）。
- M1 夠快也夠便宜。角色輕微漂移在 M1 是可接受的。

## 升級路徑 — Flux + 每角色 LoRA
- 當合作夥伴開始付費、且需要嚴格主角一致性時。
- 工具：fal.ai 或 Replicate 代管 Flux + LoRA 訓練。
- 每本書的主角從少量種子生成圖訓練一個小型 LoRA。
- 不改變渲染器與打包器就可以切換進流程。

## 風格組
- 以合作夥伴可分支的模板出貨（水彩、扁平卡通、3D 玩具、鉛筆素描等）。
- 每個風格 = 一段 prompt 片段 + 建議參考圖 + 色彩組。
- 合作夥伴選一或多個風格，可再微調 prompt 片段。

## 成本控制
- 每本一張參考圖 + N 張頁面生成。
- 安全時依 `(prompt_hash, style_hash)` 做快取。
- 失敗生成最多重試 2 次；記錄並在合作夥伴分析中呈現。
